# Ball Balancing Robot

## Overview
This is my ball balancing robot project.

I built it mainly for fun and for the learning experience.

The robot’s only purpose is to balance a ball on a glass/acrylic platform. It might seem simple, but it actually requires tight integration between computer vision, inverse kinematics, PID control, and motor control systems.
